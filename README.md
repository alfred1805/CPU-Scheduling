# CPU-Scheduling
CPU Scheduling Simulator
A Python-based command-line tool that simulates common CPU scheduling algorithms.

How to Use the Program
Run main.exe directly, no installation required.
Enter the number of processes when prompted. Fill in the following for each process:

Process ID (e.g., P1, P2, P3)
Arrival Time
Burst Time
Priority (lower number = higher priority)

Enter the Time Quantum if using Round Robin or Priority with Round Robin.
Select a scheduling algorithm from the menu:

First-Come, First-Served (FCFS)
Shortest Job First (SJF) — Non-preemptive
Shortest Remaining Time (SRT) — Preemptive
Round Robin (RR)
Priority Scheduling — Non-preemptive
Priority Scheduling with Round Robin

Run the simulation and view the results:

Gantt Chart — visual execution timeline
Table — Waiting Time (WT) and Turnaround Time (TAT) per process
Averages — Average WT and Average TAT
